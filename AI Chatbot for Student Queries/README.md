# AI Chatbot for Student Queries

## Overview
AI Chatbot for Student Queries is a simple AI-powered chatbot designed to answer common student questions such as course details, timings, fees, department information, and contact support. This project helps automate student support and improve response efficiency.

## Features
- Answers student-related questions
- Provides course and department details
- Responds to fee and timing queries
- Simple and user-friendly interaction
- Fast automated responses

## Technologies Used
- Python
- NLTK
- JSON
- Rule-Based NLP
- Tkinter / Console Interface

## Project Structure
AI-Chatbot-for-Student-Queries/
│── chatbot.py
│── intents.json
│── README.md

## How It Works
1. The user enters a question.
2. The chatbot processes the text input.
3. It matches the query with predefined intents.
4. The best response is selected.
5. The chatbot displays the answer.
